## 1. Download OCCode

Grab the latest release for your platform:

<CardGroup cols={2}>
  <Card title="macOS" icon="apple" href="https://github.com/damoahdominic/occ/releases/latest">
    Download the `.zip` for macOS (Apple Silicon & Intel).
  </Card>
  <Card title="Windows" icon="windows" href="https://github.com/damoahdominic/occ/releases/latest">
    Download the `.exe` installer for Windows.
  </Card>
</CardGroup>

<Tip>Linux builds are coming soon. In the meantime, you can [build from source](https://github.com/damoahdominic/occ).</Tip>

## 2. Sign In

When you open OCCode for the first time, you'll be prompted to sign in.

<Steps>
  <Step title="Click 'Sign In'">
    OCCode opens your browser to `occ.mba.sh` — the authentication portal.
  </Step>
  <Step title="Choose your method">
    Sign in with **Google** (one click) or create an account with **email + password**.
  </Step>
  <Step title="Redirected back to editor">
    After authentication, the browser redirects via `occode://openclaw.home/auth` which opens OCCode and logs you in automatically.
  </Step>
</Steps>

<Info>
  Every new account gets **$5.00 in free credits** — enough for thousands of AI requests.
</Info>

## 3. Start Coding

Once signed in, OCCode works like VS Code with AI superpowers:

- **AI Chat** — Open the sidebar chat to ask questions about your code, generate functions, or debug errors.
- **Inline Edits** — Select code and use the inline AI to refactor, explain, or extend it.
- **Code Generation** — Describe what you want in natural language; OCCode writes the code.

All AI requests are automatically billed against your credit balance. No API keys to configure, no environment variables to set.

## 4. Check Your Dashboard

Visit [occ.mba.sh/dashboard](https://occ.mba.sh/dashboard) to:

- View your **credit balance**
- See **usage history** (every request, tokens used, cost)
- **Buy more credits** ($5 / $10 / $20 / $50)

<Card title="Open Dashboard" icon="chart-line" href="https://occ.mba.sh/dashboard">
  View your balance, usage, and manage billing.
</Card>
