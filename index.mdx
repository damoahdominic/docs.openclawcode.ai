## What is OpenClaw Code?

OpenClaw Code (OCCode) is an open-source code editor built for AI-first development. It's a fork of [Void](https://github.com/voideditor/void) — itself a VS Code fork with a built-in AI coding agent — extended with OpenClaw's infrastructure for authentication, credit-based billing, and managed inference.

You write code. OCCode handles the AI.

## Architecture

OCCode is composed of three parts:

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   OCCode Editor  │────▶│   occ.mba.sh    │────▶│  LLM Provider   │
│   (Desktop App)  │     │   (API Gateway)  │     │  (via LiteLLM)  │
└─────────────────┘     └─────────────────┘     └─────────────────┘
     Your machine            Auth + Billing          AI Models
                             Credit Deduction
```

| Component | Description |
|-----------|-------------|
| **OCCode Editor** | Desktop app (Windows & macOS). VS Code-compatible with built-in AI chat, inline edits, and code generation. |
| **occ.mba.sh** | API gateway. Handles authentication, credit wallet, Stripe billing, and proxies inference requests. |
| **LiteLLM** | Inference router. Routes requests to the best available model via OpenRouter. |

## Key Features

- **No API keys required** — sign in and start using AI instantly
- **$5 free credits** on signup — enough for thousands of requests
- **Pay-as-you-go** — top up $5 / $10 / $20 / $50 when you need more
- **OpenAI-compatible API** — standard `/v1/chat/completions` endpoint
- **Open source** — editor and backend are both on GitHub
