## The problem

OpenClaw is an incredible AI agent runtime — but setting it up traditionally requires:

- Installing Node.js and dependencies
- Editing JSON config files
- Managing API keys in environment variables
- Running terminal commands
- Debugging connection issues manually

That's fine for developers. But **most people aren't developers** — and they shouldn't need to be.

---

## The solution: Cursor for OpenClaw

OCCode is to OpenClaw what **Cursor is to coding** — a polished desktop app that wraps a powerful engine in an interface anyone can use.
